Lohnsteuer-Demo (BMF-orientiert)

Umfassendes Eingabeformular nach Vorbild des BMF-Rechners 2026. Die Beträge werden überwiegend in EUR erfasst, in Cent umgerechnet und lokal über ./dist/core/index.js berechnet.

Ohne npm-CDN-Abhaengigkeit

Eingabeparameter

Feldbezeichnungen folgen, soweit sinnvoll, den PAP-Parametern.

Allgemeine Besteuerungsmerkmale
Angaben zum Bruttolohn
Vorsorgeaufwendungen und Versicherungen
Freibetrag und Hinzurechnung je Zeitraum
Erweiterte Eingaben (Sonstige Bezuege, Jahreswerte, Versorgungsdaten)
Bereit.

Ergebnisse

0
0.00 EUR
SOLZLZZ
0
0.00 EUR
Ausgabe Cent EUR

Generierter API-Payload

Genau dieses Objekt wird an calculate(year, inputs) übergeben.

{}

Anwendungsbeispiel

import { calculate } from "lohnsteuerrechner";

const result = calculate(2026, {
  LZZ: 2,
  RE4: 500000,
  STKL: 1,
  KVZ: 2.5,
  PVZ: 1,
});

console.log(result.LSTLZZ);

Angelehnt an das offizielle BMF-Eingabeformular: bmf-steuerrechner.de.
